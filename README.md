# Gerenciador de Tarefas em Java 

Projeto desenvolvido como parte do meu portfólio de Engenharia de Sistemas. É um sistema de linha de comando (CLI) focado em Programação Orientada a Objetos e persistência de dados.

## Tecnologias e Conceitos Utilizados
- **Java 21**
- **Maven**: Gerenciamento de dependências e estrutura do projeto.
- **POO (Programação Orientada a Objetos)**: Uso de classes, encapsulamento e coleções.
- **Persistência de Dados**: Leitura e escrita em arquivos `.txt` (I/O).
- **Git/GitHub**: Controle de versão e boas práticas com `.gitignore`.

##  Como Executar
1. Clone o repositório.
2. Certifique-se de ter o JDK instalado.
3. Execute a classe `App.java` através do VS Code ou via Maven.
